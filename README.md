## Getting Started
  
### Prerequisites

All you need to do is run yarn after cloning the project and you will be set.

* yarn

```sh

yarn install -g

```

  

### Installation

1. Clone the repo

```sh

git clone https://github.com/McTrip777/AllSortingAlgos.git

```

2. Install YARN packages

```sh

yarn

```
